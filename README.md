# ShawnyOS Portfolio

![Version](https://img.shields.io/badge/version-1.0.0-blue.svg)
![License](https://img.shields.io/badge/license-Proprietary-green.svg)

A dynamic web-based OS-style portfolio platform that showcases the power of JavaScript through an interactive, customizable desktop environment. ShawnyOS combines a traditional portfolio with a fully functional app ecosystem where users can install apps, customize their experience, and explore creative projects.
